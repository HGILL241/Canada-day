CANADA DAY CROSSWORD SPRINT - SETUP

Files included:
- index.html
- style.css
- script.js

HOW TO TEST:
1. Open index.html in your browser.
2. Play the game.
3. Leaderboard works locally on the same device by default.

FOR TRUE LIVE LEADERBOARD ACROSS ALL PHONES:
1. Go to Firebase Console.
2. Create a new project.
3. Add a Web App.
4. Enable Realtime Database.
5. Copy your Firebase config into script.js.
6. Change USE_FIREBASE from false to true.
7. Host the folder using Netlify, Vercel, GitHub Pages, or Firebase Hosting.

QR CODE:
Once hosted, copy the live URL and generate a QR code using any QR code generator.
Use the QR code on posters, slides, or Teams announcements.
